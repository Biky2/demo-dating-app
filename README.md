# Flutter Dating App – Advanced Clean Architecture & Riverpod

A modern **Tinder-style dating app** built with **Flutter**, showcasing **Clean Architecture**, **Riverpod state management**, and a **repository pattern**. Fully local, optimized for performance, and responsive across all devices. Ideal for portfolio, learning advanced Flutter, or testing scalable architecture patterns.

---

## 🚀 Features

- **User Authentication**  
  - Local email/password login  
  - Persisted login state  

- **Swipe Cards**  
  - Tinder-style swipe interface  
  - Swipe right to like, left to dislike, up for premium super-like  
  - Local match system  

- **Profile Details**  
  - Hero animations for smooth transitions  
  - Premium badge support  

- **Matches & Chat**  
  - View matches  
  - Local chat simulation with typing animations and timestamps  

- **Subscription System (Fake)**  
  - Free tier: limited swipes  
  - Pro tier: unlimited swipes and super-like  
  - Upgrade stored locally  

- **Multiple UI Modes**  
  - Light Mode: minimal & clean  
  - Performance Mode: optimized for low-end devices  
  - Premium Mode: smooth animations & advanced UI  

- **Responsive Design**  
  - Fully responsive on phones and tablets  
  - Adaptive font and layout scaling  

---

## 🛠 Technical Highlights

- **Flutter 3.x** with Material 3  
- **Riverpod** for state management  
- **Clean Architecture**: domain, data, presentation, core  
- **Repository Pattern** for decoupled logic  
- **GoRouter & ShellRoute** for nested & persistent navigation  
- 50 dummy users + 5 fixed test login accounts  
- Optimizations: `const` constructors, lazy loading, cached images  

---

## 💡 Why This Project?

This app demonstrates advanced Flutter concepts:

- Scalable state management  
- Clean and maintainable architecture  
- Multi-device responsive UI  
- Animations and smooth transitions  
- Realistic app flow without a backend

## 🏁 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/Biky2/demo-dating-app.git

```
2 .Navigate to the project:
```bash
cd flutter-dating-app
```
3.Install dependencies:
```bash
flutter pub get
```
4.Run the app:
```bash
flutter run

```
